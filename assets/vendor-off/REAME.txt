[ABOUT]

O "Typing Jekyll Template" usa esté diretório para desenvolvedores do projeto, porém, você pode manter para o projeto final também, caso queira seus arquivos de fornecedores dentro do Typing Jekyll Template". Isso vai fazer com que você possa gerar "Typing Jekyll Template" normalmente sem ter uma conexão com a internet, já que uma vez, o "Typing Jekyll Template" usa fonecedores online.

Este diretório, irá amarzenar os seguintes arquivos de fornecedores:

* JQuery
* Bootstrap
* Font Awesome
* Fonts

[USING]

Para usar o recorder de desenvolvimento offline (vendor-off), deixe valor "vendor-off" no arquivo "_config.yml" para 'true'.
