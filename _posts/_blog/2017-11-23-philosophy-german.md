# 140227上

现代性:现代化。文化世界观的总体状态。

十八世纪末，启蒙运动本身的规律性昭然若揭。

启蒙运动基本原则:

1. `理性批判`，批判教会。

2. `科学的自然性`，而非神圣的东西。

文化的本意是你有没有一颗柔软的心。

当下主流思想:`世界是有规律的。掌握规律了就可以改变世界。`

十八世纪末，发现不对:

1. 理性批判导致`怀疑主义`，瓦解了外部世界和自己的信仰。英国经验论:怎么证明，你看到的世界是世界本身而不是你的世界？对其它心灵的信念。我认为你开心，可我不是你怎么知道的？一切坚固的东西都土崩瓦解了。

2. 科学自然性导致`唯物论`，自由，不朽，精神都没了。所以抓住当下。享乐主义。精神是脑细胞的运动。？反讽是人特有的技术？诗意？双关？电脑做不出来的。

# 140227下

康德是启蒙的集大成者，是维护启蒙的。但他对自然主义有强烈的批判。德国观念论从康德到黑格尔，是为了`批判和拯救启蒙`。

启蒙是理性时代:

1. 理性是一种`批判`的能力。休漠，如果没有基本的信念，一天都活不下去。自然信念。理性要求拿出证据来。

2. 理性是一种`解释`的能力。自然科学。所有事物都是一个规律的特例。

当今受过教育的人出发点就是以上两点。

启蒙的危机为，`不得不把这两点推到极致`。限制理性就是非理性。对他人心灵的信念来自于他人的表现自己的推断。因为没有手段了解和确证。休漠：`我怎么知道我，我就是一连串感觉。`我们的感觉是不够的。理性推倒了他人，他为自己却找不到根据。

康德批判启蒙这个危机的时候，拯救启蒙。哥白尼似的革命。对理性要有批判，康德第一批判就是做这个。选择没有错，操作有错。

马克思、尼采、弗洛伊德，现代思想三大先知。

人是理性的动物，是可笑的。人是本能的动物。

康德，还有其他因素，我们认识对象受制于我们的认识方式。之前人没考虑认识的主观思路。

彻底的自然主义的唯物论。科学可以解决人舒适的问题，但解决不了人生观的问题，人怎么活的问题。人是水，是物质，没有神圣性。契科夫，托尔斯泰，雨果。

唯物主义二元论，物质，精神。线性因果论。

# 20140304上
观念是把握世界不可少的途径和媒介。说明不了精神是什么。启蒙两大支柱，理性批判和科学互相拆台。英国经验论。理性批判最后把自然科学主义给挖掘了。因果律，没有客观必然性，只有主观或然性。打雷下雨重复，产生了心理联想而已。笛卡尔霍布斯洛克休漠都在这个地方有矛盾和疑惑。
十八世纪末德国，开始了元批判，对启蒙运动批判的批判。哲学以外的考虑，发现近代科学和理性导致道德的衰败。功利和实用主义道德观。私人财产产生，导致贞洁观。宗教，国家的衰弱。无神论，无政府主义。
# 20140304下
宗教是处理感情的。宗教是必须要有的。康德代表不妥协的哲学批判，斯宾洛莎代表彻底科学的自然主义。但后果是，如果把他对物自体的设定放在一边的话，实际上是唯我论。斯宾罗莎的哲学，如果去掉表明宗教成分，实际上是无神论和宿命论。经过启蒙熏陶的人们，不能再回到中世纪。但他们也知道信仰也不能丢。理性的疑和信仰的信如何选择与平衡？德国人雅克比引进虚无主义，休漠就是虚无主义者，唯一肯定的是无本身。海德格尔说，西方形而上学的本质就是虚无主义。理性无条件的批判应该也适用于批判本身。康德的批判哲学就是这个。批判的批判，元批判。纯粹理性批判，实践理性批判，判断力批判。理性无偏见，是假像，工具理性是受利益的欲望支配的。德国哲学家哈曼，反对理性纯粹性的主张。事实上理性和文化，语言，经验分不开的。理性有他的社会和历史纬度。理性的工具是语言，而语言体现了一个民族的文化和传统。理性是意志的工具，是受意志指导的。理性是自然的一部分，则理性不再是自主的，离不开自然规律，最终导致自然主义。康德来解救这个矛盾。
# 20140306上
叔本华思路上是康德往下走。康德的成就来自于他伟大的人性，而不是聪明。康德说，哲学是道德学，他是道德哲学家。至善是纯粹理性批判的终极目的。纯粹理性批判，康德说，哲学问题集中在以下三个方面:
我能够知道什么？(认识的范围)
我应当做什么？(道德)
我可以希望什么？(人本性和希望的关系？)
康德逻辑学讲稿第四个问题:人是什么？(以上三个问题的归结)。你是什么样的人，就有什么样的哲学。康德(1724-1804)，格尼斯堡(加里林格勒)。
# 20140306下
1770年，康德近五十岁。纯粹理性批判之前，前批判时期，以后，批判时期。《未来形而上学导论》《道德形而上学》比较容易看些。《永久和平论》。康德提出批判哲学时所面临的问题:
是他面对启蒙哲学的理性进行批判。让理性有界限，有它做不到的，有他不该管的。”我要限制知识，给信仰留下地盘。“正义，不是理性的问题，而是良知的问题，良知和信仰挂钩。“理性水穷处，信仰云起时。”“休漠让我从独断论中惊醒。”休漠之前，理性是上帝赋予的。休漠说不对，因为人和世界接触是经验，经验特点是应人而异，而且经验看到的事物不是事物的本身。康德同意休漠，人无法认识事物本身。但康德不同意休漠，认为人并不是不可能有普遍有效的知识。而休漠认为，比如因果性，是不可靠的。
唯物主义导致人生意义的丧失。消费主义，享乐主义。
康德要建立没有怀疑论的批判(抵抗笛卡尔和休漠怀疑论对外部世界知识的说明)，没有唯物论的自然主义(坚持自然事物可以用力学解释，但不认为一切存在的东西都在自然中。有超自然的东西，我们不能认识不代表它不存在。黑格尔借重的是生命有机科学，康德借重的是古典力学)。
他要化解理性批判和自然主义的冲突，他要创造一个避免自然主义危险的批判(理性是自主的，而不是物质的一种功能，不能让唯物论进来)，和没有怀疑论危险的自然主义(物理学规律只适用于自然本身，自然规律不是休漠说的我们心理联想的习惯，而是客观规律，是理性构造的，但可以用于自然世界)。
自由是什么？自由问题不是物质问题(一般说法:人类作为大自然的一部分，服从自然规律，是没有自由的)，也是超越理性批判以上的问题(也服从社会规律，自由是相对的不是绝对的)。排除以上两个错误的方法。自由和人的尊严的相关性彻底拔掉了。

# 20140311上
形而上学想对世界有个整体的把握。启蒙让形而上学陷入了危机。我能做什么？我应该做什么？我可以希望什么？解决这些问题的根本途径是对理性进行批判。没有办法回答的问题有没有价值？

# 20140311下
严守分际。康德真正的贡献，是批判采取的特殊形式。把外在的真理观颠倒过来。康德认为，真理应该是客观知识和主观的符合，而不是主观符合客观。想要通过概念，先天地构成有关对象的动机，以获得我们知识的一切努力的尝试都失败了。比如因果性，就是先天的，我们设计的概念结构。指的就是休漠。因此不妨试试倒过来，让对象符合知识。康德的表象，指观念，范畴，概念。一般真理观，在经验本身可能性的说明，会有问题。经验，如果只有感觉，没有观念，是不可能的。比如感觉的经验，都是有序的，这个序哪儿来的？经验是有先天条件的。不是我们创造了对象，而是我们能认识它，是因为它符合了若干条件。比如，我们感知世界要通过时空，所以事物向我们呈现的必定在时空之中。康德认为现象和事物本身区分的二元论。哲学对付的是现象。后人认为这是康德的败笔。先验是不依赖于经验，而不是先于经验。不依赖于经验并不是和经验没有关系。闭门造车，出门合辙。是经验成为可能的条件。存在论问题被排除了，怀疑论的问题在于外在的真理观。

# 20140313上
真理观思路倒过来了，所以被称为哥白尼的革命。现象是实在的。实在和我们普遍必然的观念形式的符合。二元论的先验哲学。康德坚持自然主义，在自然界发生的一切符合自然律。自然律只对现象有效。事物本身我们是没有办法把握的。区分现象和本身，理性表明自然律适用于所有自然现象。美的形式是美的性质的条件。

# 20140313下
柏拉图的形式，但没有说人和形式怎么互动的。柏拉图说，在那个世界，我们形式和灵魂是统一的，来到这个世界，由于受到污染，忘掉了，所以要把以前有的给回忆起来。学习哲学就是演习死亡。善的理念照耀着。实在论，人的思想能够发现事物的本质。人可以超出自己的经验。唯理论。怀疑论，没有客观知识，我们认识的只是感官知觉的知识。内在性，我们只能有经验内部的知识。经验论。康德哲学是两者的取舍。对怀疑论肯定的是内在经验，经验以外的东西不可知。对独断论肯定的是它的唯理论。他通过他的二元论综合了独断论和怀疑论。肯定了知识只能在经验领域。二者共同的错误是，误解了经验对象的定位，以为他们本身是他们对我们显示的那样，“赤裸裸的经验”，没有掺杂人的主观构造因素。康德认为，现象世界，既是感性的也是可知的，而本体世界两者都不是。对于现象而言，怀疑论认为不可能有知识，是错的，我们可以有关于经验对象的先天知识。关于本体，独断论是错的，认为有关于物自体的知识，康德认为不可能。他把形式和理念改造成范畴。范畴有主观建构的意思，而亚里士多德的形式和柏拉图的理念都没有这个意思。
康德把思维和对象的关系倒过来，传统柏拉图主义者认为对象刺激思维，而康德认为物我不分的。是我们的思维”构造”对象，形成对象的形式关系意义规定。范畴是构造对象的规则。不是乱构造的。是普遍有效的。知识是判断组成的系统，定义的勾连。由于对象之为对象，已经被构造过了，我们说的是纯范畴，是先天的，不是经验的归纳。所以它构造的对象有可能形成先天的知识，不是对象的归纳。比如几何学。如何证明先天知识是有的？怀疑论认为根本没有。独断论认为有，但没办法解释怎么来的，怎么会有效？

# 20140318上
知识是判断体系。先天分析判断:三角形有三条边，三角形意味着有三条边，对知识没有增加，是同意反复，是永真的。后天综合判断，谓语所说的并未包含在主语中:张三一米六，否定它是不矛盾的，没有普遍必然性。
如何证明判断？后天证明，诉诸于经验，充满了或然性。先天证明，如数学，逻辑，有普遍必然性。休漠说，人类有两种知识:观念的知识，经验的知识。物理化学不是观念的知识。康德要回答休漠，他提出，先天综合判断，材料来自于经验，但普遍必然性来自于人的主体性，有效范围只在现象，而不是事物本身。灵魂是不朽的，先天，因为无法经验论证，综合，因为谓语不包括在主语中。数学，7+5=12，经过操作得到7,同样操作得到5，12是永真的，所以是先天的，又增加了我们的知识，是综合的。他这个观点今天依然很多人不同意。

# 20140318下
如何证明先天综合判断？形而上学的问题也可以这么证明吗？我们的经验是由两种不同的表象结合的:概念和直观。经验不是来自于事物本身，而是概念(Begriff)和直观(Anschauung)的结构。概念和直观是两种不同类型的表象。一块石头，分为:个别使用的石头，直观。普遍属性的石头，概念。直接意识到个别事物，间接意识到普遍属性。判断是对象的表象的表象，是直观和概念的表象的结合。数学里无限的概念，没有感性直观，但数学是先天综合，所以我们需要空间的表象。我们对空间和时间必须要有纯直观。不是理性也不是经验直观。时空是我们先天直观形式。时空是观念性的东西，而不是感性的。牛顿认为时空是外在的，康德说时空是我们内心的结构，是经验事物的基本条件。时空从起源来讲是主观的，属于我们人的，但普遍有效性来说是客观的，不为尧存不为桀亡。这里的客观意思是:客观是主观之客观，而不是马列说的事物本身。
仅仅凭直观，无论感性直观还是观念性直观都得不到关于事物的任何知识。知识需要有概念的活动。知识需要判断，对事物结构的联系。我们的意识，就是这两者（直观和概念）的结合。我看到一块石头，是被动的，我们想这个石头很重，是主动的。
表象和它所表象的对象的关系？感性直观只能提供我们不确凿，杂多(有一定规则)的经验。但杂多的经验不可能是完全无序的方式对我们呈现的。对象(Gegenstand)是概念和直观以一种有规则的方式结合在一起。

# 20140320上
不是因果问题，是规范性问题。两类:事实性的描述(今天是周四)，规范性的描述(今天是好天。因为好坏不是事实，是人设定的一种规范)。判断一个现象的真假，涉及到规范性。真假是规范性的术语，而不是事实的描述。规范性事关我们应该如何对待事物，而不是我们事实上如何对待事物。我们判断一个经验对他的对象来说是真还是假，要区分三件事:？？习惯的秩序。经验和对象也有不符合的问题。既然对象是我们的经验之构造，为什么还会不对呢？经验有两部分，一个是我们直观的能力，还有一个是被我直观到的对象。
对待经验，包括三个步骤:
首先，我们必须以统一的方式把握直观对象，这样经验的杂多(Mannigfaltige/Mannigfaltigkeit)就可以作为一个时空框架当中一个确定的东西，为我们在那里，以便让我们对它做出判断。课桌在哪儿都是课桌，因为我们经验里面勾连它的规则是一致的。
第二，我们必须根据某套规则，统一直观到的经验的事物，这样我们的经验将显示某种规则性，那么我们就可以根据这套规则来判断杂多。
第三，我们必须把直观归摄(subsumieren)在概念下。因为概念是把表象组织在一起的规则。必须让直观表象服从规则。必须达到第三步，否则第一第二部讲不清的，规则就不清楚了。判断(Urteil)本身是个表象，判断这个表象与判断这个表象的对象之间的关系是什么？伴随着我们的所有的表象，既有表象，一定有他(思维的主观的操作者，运用规则的超盘手 )，因此直观一定会被归摄在概念下。除非我能够意识到一个表象就是一个表象(石头就是石头，水就是谁，树就是树)，否则对我们来说表象就是无。因此任何表象都是表象的条件，他有某种认识内容。表象的条件和自我意识本身的条件是有关的(表象跟我们人的构造能力有关)。
统觉(Apperzeption)就是能够把事物的条件和自我意识的条件合在一起。统觉的本质是什么？统觉就是任何一个把杂多表象为杂多的表象。即使是杂多经验为杂多，也有单一的综合的思维在里面。哪怕被外力推了一下，这个表现里面也有综合，否则连被推这个表象都不会有。经验杂多的统一，就是表象的综合统一。单一的主体，表象的杂多，主体显然和表象的杂多的综合统一是有关联的。表象杂多的综合统一的条件，是统觉具有综合统一的能力。我们的主体，我思(cogito)，不可能只有单一的经验，有无数的经验。我所有的经验里面都有一个同一的我思，它既是必然的又是综合的。康德的判断理论，没有判断就没有知识。

# 20140320下
康德第一次区分了经验主义和先验主义。他突破了经验论者(如笛卡尔)由于没有区分经验和先验而造成很多困难。
判断就是让自己服从做判断的规范。错误的经验，是把规则用错了，没有用大家公认的方式勾连表象杂多。做判断是是一个事物服从某种正确性的标准(把某个观念和某个观念联系在一起的规则)。判断本身是个规范性的事情，是根据正确的判断规则把两个不同类型的表象连接为一个统一。
这又表明概念不是简单的从直观当中的抽象。概念(Begriff) 是判断进行综合的规则。概念不是共名。‘数’是个共名，不是个概念。“概念把各种表象置于一个共同表象下(归摄)行动的(意味着综合，不是死的)统一。”有一个概念就是有一个综合判断的规则。概念是一种能力，一种根据某些规范结合表象的能力，而不是人的主观的内心的精神状态。
康德把这些最最基本的概念叫做范畴(Kategorie)。范畴是一般对象的概念，通过这些概念，对象的直观，就在判断的逻辑机能(事物之间必然结合的规则)的某个方面看做是被确定的(对象的直观，这些杂多只能这样结合，而不是N种方式结合)。没有这样的范畴，我们就不能把我们的直观看作是“表象”，他们充其量只能是“心理事件”。康德的“表象”有普遍必然因素“在”，而“心理表象”没有这个“在”，因为他不确定，所以没有对错之分。客体(Object)就是在其概念中结合的一个锁机，锁于直观的杂多的那种东西。客体就是通过概念，把所给与我们的直观的杂多结合在一起。可以看到这个客体是主观构建起来的，而不是原来说的客体就是在那里的存在。直观本身不能告诉我们他们是什么(因为我们看到的是椭圆形，长，颜色等)，是我们根据概念表象的规则，主动地把这些表象结合在一起，这样他们才成了某物的直观(石头的直观)。他们之所以有规定性，是因为我们按照某种特定的规则来这样组织他们的，把他们结合在一起，他们才是“某物”。他们是什么，是我们给予他们的。直观也会有错。比如说石头，你说他不是石头，不是你看错了，就是你规则使用错了。统一在一个概念中的表象可以使感性直观或者其他概念。感性直观是直接的，概念是中介性的，除了单纯的直观外，没有任何的表象是直接取自于对象的。所以概念永远不和对象直接发生关系，而是和关于对象的某个表象发生关系。判断就是对对象的间接知识，因而是一个对象的表象的表象。判断和对象都是综合的，综合的意思是经过建构和结合的。范畴是根据判断的类别来的，有多少种判断的形式，就有多种范畴。亚里士多德有四大类十二种判断，所以康德认为范畴也分为四类十二种范畴。黑格尔批评这个太粗糙，应该根据思维的理论来得出到底有那些范畴。基本的范畴本身就得与我们把我们的感性经验安排结合为一个统一的经验的方式。关系到我们如何来安排自己的感性经验，如何来结构自己的感性经验，把他们结构成一个统一的经验的方式。范畴的重要就在这里。统一的经验表现为单一的世界。
心灵不能被理解为被动的东西。我们不只是处于和客体的因果关系当中(他在那里-因，我看到了-果)。我们和对象的关系，是我们通过把经验当中的各种因素(包括直观的和概念性的)结合在一起，这是一种积极的而不是消极的方式，是主动的而不是被动的方式。
罗蒂《哲学和自然之镜》，几千年的西方哲学的最大问题就是认为知识是一面镜子，是对世界的反映。他认为这是根本的错误。他的立场是海德格尔和加德摩尔的立场：世界是由意义构成的，而不是由事物构成的。事物本身是意义。事物本身是没有的，要看我们对他的意义的理解。因为你把它当成什么就是什么。是康德走出了关键一步，把事物本身给勾掉了。普遍旧观点认为所有东西都是事物的反映，那么同一个事物为什么在镜子里面照出来有不同的？事物本身不能决定任何事物的表象。表象之为表象是我们对他动了手脚了。康德认为自然事物本身是有的，但是它自己无法成为表象，他要成为表象必须由我们对他进行加工和构造，它才能成为表象向我们呈现。他们只是存在着，但只能以我们能够接受的方式以表象出现。我们的感性直观只是通过与被非直观的感性形式结合在一起而成为客体的表象。离开概念与直观的结合，概念是纯粹的空洞形式的规则。“思维没有内容则空，直观没有概念则盲”。
但要注意，表象不是私人的内心活动的产物，而是我们先验主体性的一个结构性的结果，不是经验心理学的结果。因为规则不是你的，而是先验主观性的规则，是对人类来说的结构性功能，和你主观动脑筋不动脑筋没关系。做判断，就是接受正确性的标准(不为尧存，不为桀亡)。这规则不是经验规则(私人规则，心理规则)，是先验规则。私人规则不是规则，只是个人的爱好，没有普遍必然性。是普遍的自我意识的，我们人作为类的规则。
我们能够认识到的对象一定是在时空之内的对象，因为时间和空间是我们任何可能的直观的形式。
